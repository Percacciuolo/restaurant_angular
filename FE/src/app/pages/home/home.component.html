App di gestione ordini per un ristorante

Obiettivo:

Creare un'app di gestione ordini per un ristorante con una pagina del menu dove i clienti possono ordinare dei piatti, una pagina di gestione delle ordinazioni lato cucina ed una pagina per modificare il menu.

Requisiti:

  1. La pagina del menu dovrebbe mostrare una lista di piatti disponibili con il nome, la descrizione, il prezzo e un'immagine del piatto.
  2. I clienti dovrebbero poter aggiungere o rimuovere i piatti dal loro ordine, e visualizzare il totale del loro ordine.
  3. La pagina di gestione delle ordinazioni lato cucina dovrebbe mostrare un elenco di tutte le ordinazioni in corso, con il numero del tavolo, l'elenco dei piatti ordinati e lo stato dell'ordine (in attesa, in preparazione, completato).
  4. Lo stato dell'ordine dovrebbe poter essere aggiornato dalla pagina di gestione lato cucina.
  5. La pagina per l'edit del menu dovrebbe mostrare tutti i piatti già presenti in menu con la passibilità di modificarli, cancellarli o crearne di nuovi

Suggerimenti:

  • Utilizzare Angular Material per la creazione dell'interfaccia utente.
  • Utilizzare un servizio per gestire la logica dell'ordine e lo stato dell'ordine.
  • Utilizzare il routing di Angular per navigare tra le pagine del menu e delle ordinazioni.

Suggerimenti avanzati:

  • Implementare la funzionalità di ricerca per i piatti nel menu.
  • Implementare la funzionalità di notifica per gli ordini completati.

Integrazione Backend

È stato integrato un server con 4 api:

  • http://localhost:4000/api/mock/getOrders (GET). L'api tornerà un array con le ordinazioni effettuate.
  • http://localhost:4000/api/mock/getMenu (GET). L'api tornerà un array con il menu.
  • http://localhost:4000/api/mock/updateMenu (POST). L'api prenderà in input un obj con una proprietà menu dove inserire l'array del menu aggiornato, tornerà l'array del menu aggiornato
  • http://localhost:4000/api/mock/submitOrder (POST). L'api prenderà in input un obj con una proprietà newOrder dove inserire la nuova ordinazione, tornerà l'array degli ordini aggiornato
Per configurare il server aprire un cmd nella cartella "mock-server" e lanciare il comando npm i. Dopo aver installato la node_modules lanciare il comando npm run start:server per startare il server. Il server starà in ascolto alla porta 400, e per interrogarlo bisognerà effettuare le chiamate (GET,POST) agli url sopra.
